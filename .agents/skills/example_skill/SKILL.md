# Instructions

當使用者要求使用 `example_skill` 時，請遵循以下步驟：

1. 確認使用者的需求。
2. 根據需求提供相關的協助。
3. 如果需要，可以讀取 `./scripts` 或 `./examples` 裡的資源（這裡目前為空）。

# Best Practices

*   保持 Skill 指示簡潔明確。
*   為每個 Skill 提供清楚的 `description`，讓 Agent 能夠正確辨識何時該使用。
