整體解決方案結構

Git Monorepo

  • 包含 Common：共用模組 (Auth, Data, Audit, Health)
  • 包含 Modules：業務模組 (Identity, Admin, AI相關)
  • 包含 Apps：產品端 Host (OfficeAI, ISAEdge)
  • 包含 Frontend：前端應用 (Vue 3 SPA)

Docker 獨立部署

  • Office AI 與 ISA Edge 各自有獨立 docker-compose 設定
  • 資料庫各自獨立操作 (PostgreSQL 實例獨立)
  • 共用開發時的 ProjectReference，發布時統一 Docker 包裝
